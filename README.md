# Automotive Predictive Maintenance System

A production-grade machine learning system for predicting component failures 
and estimating Remaining Useful Life (RUL) from automotive telematics data.

This project simulates real-world BMW use cases such as:
- Battery health forecasting
- Brake system degradation prediction
- Engine vibration anomaly detection
- Remaining Useful Life (RUL) estimation
- Real-time telemetry monitoring & alerting
